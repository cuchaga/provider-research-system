## What is Provider Research System?

A multi-skill AI-powered system for researching healthcare providers with intelligent orchestration, semantic matching, and web scraping capabilities.

## Features

- **Natural Language Understanding** - Ask questions in plain English
- **Multi-Skill Architecture** - Coordinated AI agents for different tasks
- **Database Integration** - PostgreSQL and SQLite support
- **Web Research** - Automated data extraction from websites
- **Semantic Matching** - Intelligent provider matching beyond exact strings
- **NPI Validation** - Integration with NPI registry
